# 오픈 이슈

## 1. delete 감지 UX
- soft delete를 first-class로 넣을지
- tombstone helper 제공할지

## 2. cursor drift
- app이 `updated_at`를 갱신하지 않으면 누락 가능
- lint/validate로 얼마나 강하게 막을지

## 3. long-running handler
- Functions timeout과 lease heartbeat를 어떻게 맞출지
- chunked handler mode가 필요한지

## 4. source query flexibility
- 사용자 raw query 허용 범위를 어디까지 둘지
- unsupported SQL 문법 경고를 어떻게 줄지

## 5. reset UX
- CLI에서 reset 권한 오용 방지
- dry-run / confirm prompt 필요

## 6. outbox helper
- 단순 라이브러리만 제공할지
- table DDL 템플릿까지 줄지

## 7. driver extras 정책
- psycopg vs asyncpg
- pymysql vs mysqlclient
- pyodbc vs mssqlpython

## 8. OpenTelemetry
- core dependency로 둘지 extra로 둘지
